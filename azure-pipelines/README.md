# Azure DevOps Pipeline Examples

This folder contains example Azure DevOps pipeline YAMLs for both Linux and Windows agents.

- `azure-pipelines-linux.yml`: Uses bash scripts
- `azure-pipelines-windows.yml`: Uses PowerShell scripts

## Usage

1. Copy the relevant YAML to your Azure DevOps repo as `azure-pipelines.yml`.
2. Commit and push changes after each step.
3. Monitor pipeline runs in Azure DevOps UI.

## Example Steps
- Print Hello World
- Show environment variables
- Check disk space
- Download a file
- Archive workspace
